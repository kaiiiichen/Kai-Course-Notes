# Security & Sensitive Content Policy

This is a course-notes archive, not a software project, so "security" here mostly means **sensitive content**:

- Accidentally published homework/exam **solutions** or OJ reference programs
- **Copyrighted** material that should not be redistributed
- **Personal information** (names, IDs, grades) appearing in scanned documents

If you spot any of the above, please report it **privately** via [GitHub private vulnerability reporting](../../security/advisories/new) (preferred), or through the contact channels on [kaichen.dev](https://kaichen.dev) — rather than opening a public issue that would draw attention to the content before it is removed.

Reports are typically handled within a few days. Thank you!
