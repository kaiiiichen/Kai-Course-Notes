# Contributing

Thanks for your interest! A few things make this repository different from a typical open-source project, so please read this before opening a PR.

## ⚙️ This repository is generated

The public tree you see here is **machine-generated** from a private superset via a sync script. Direct pushes to files are periodically overwritten by the next sync. Because of that:

- **Prefer opening an Issue** (broken file, wrong content, dead link, typo) over a pull request — I will fix it at the source so the fix survives future syncs.
- Small PRs are still welcome as *signals*; I may apply your change at the source and credit you, rather than merging the PR directly.

## 🚫 What will not be accepted

- **Homework, lab, or exam solutions of any kind** — this repository deliberately excludes them for academic integrity, even in PRs and issues. Please do not paste solutions into issue text either.
- **Published textbooks or other copyrighted files.** Course READMEs list reference books instead.

## 📢 For lecturers / rights holders

If you would like your material removed, please open a **Takedown request** issue (template provided) or contact me through the channels on [kaichen.dev](https://kaichen.dev). Removal requests are honored promptly, and per the repository license note, all prior downloads of the removed material are no longer authorized.

## 🗂 Layout

`<University>/<Course>/…` — each course folder has its own README. Web-native notes live on the [Notion branch](https://kai-course-notes.notion.site/main).
